# Manual ILC – Construcción de GPTs Empresariales

# Capítulo 1. Bienvenida y Caso de Negocio

Bienvenido al Manual ILC. Este documento te acompañará paso a paso para construir un GPT empresarial, incluso si nunca antes has creado uno.

## ¿Qué aprenderás?

- Comprender un problema de negocio.
- Transformarlo en un Caso de Uso.
- Diseñar un GPT Canvas.

- Configurar un GPT en ChatGPT.
- Validarlo y mejorarlo.

## Caso guía

Durante todo el manual utilizaremos el caso **ILC-16 – GPT Estructurador de Requerimientos Negocio–Tecnología**.

### El problema

Un analista de Tecnología recibe solicitudes por correo, reuniones o chats. Con frecuencia estas solicitudes son incompletas y generan retrabajo.

Ejemplo:

> "Necesitamos mejorar el módulo de clientes porque está muy lento."

Con esa información no es posible desarrollar una solución.

### ¿Cómo ayuda un GPT?

El GPT actúa como un Analista Funcional. Antes de proponer una solución realiza preguntas, identifica información faltante y organiza el requerimiento.

### Beneficios

- Reduce ambigüedad.
- Disminuye retrabajo.
- Estandariza requerimientos.
- Mejora la comunicación entre Negocio y Tecnología.
